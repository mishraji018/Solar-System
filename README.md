# 🌌 Solar System Simulation

A **3D Solar System Simulation** built using **Three.js, HTML, CSS, and JavaScript**.
This project visualizes planets orbiting the Sun with smooth animations and interactive zoom controls.

---

## 🚀 Features

* ☀️ Sun placed at the center with glow effect
* 🪐 All **8 planets** orbiting the Sun
* 🔄 Planet **rotation and revolution animations**
* 🖱 **Mouse scroll zoom** (zoom in / zoom out)
* 🌠 Starry space background
* ⚡ Smooth animation using `requestAnimationFrame`

---

## 🪐 Planets Included

* Mercury
* Venus
* Earth
* Mars
* Jupiter
* Saturn
* Uranus
* Neptune

Each planet revolves around the Sun with **different orbit speeds and distances**.
